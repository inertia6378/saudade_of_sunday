NodeJs

What is NodeJs?

Node Js is a JS runtime. It's commonly used as a wrapper around a VM. There are lot of benefits for using node below. The gist is that the built in packages allow for high level application design. It also ships with NPM for dependency management, something long over due for JS. You can also create C++ addons to increase functionality at high speeds. Node is a great platform for tools. You have many options out there as Node ships with NPM as a package manager. Node allows you to use JS for your entire stack.

  1. Wrapper for V8
  2. Built in Modules
  3. Asynchronous Api's with no Threads
  4. C++ Addons
  5. Debugger and utilities
  6. Module Dependency Management
